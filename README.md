# BootAnimation.zip-PREVIEWER
Androidのbootanimation.zipをプレビュー再生できるPythonスクリプト

# 機能
Partの詳細表示
desc.txtの読み込み
再生、一時停止、再生位置変更
trim.txtの読み込み
FPSの変更
端末ごとの解像度の変更(チャレパに最適化済み)
ドラッグアンドドロップでの再生
キャッシュ生成でスムーズな再生
# インストール方法
pythonを環境変数に入れてインスコ後、
Bootanimpreviewer.pyをダウンロードし、
cmdにて

```python -m pip install PySide6 Pillow```

または


```pip install PySide6 Pillow```
を実行。

# 実行方法
bootanimpreviewer.pyがあるパスでcmdを開き、
cmdにて
```python bootanimpreviewer.py```
を実行するだけ。
